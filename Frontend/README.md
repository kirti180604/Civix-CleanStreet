# CleanStreet — Login Page

This is a small React + Vite skeleton implementing the login page (desktop split layout) based on the provided design.

What's included
- `src/Login.jsx` — the login form component
- `src/styles.css` — styles to match the screenshot (split layout, warm brown button)
- `index.html`, `src/main.jsx`, `src/App.jsx`

Run locally (Windows PowerShell):

```powershell
# 1) Install dependencies
npm install

# 2) Start dev server
npm run dev
